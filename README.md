# Todo App Frontend
### Todo App Frontend sử dụng ReactJS
## Yêu cầu
-   NodeJS
## Hướng dẫn cài đặt và sử dụng
-   Tải các phụ thuộc cần thiết cho dự án bằng lệnh dưới dây ở trong thư mục của dự án
```
npm install
```
-   Chạy server bằng lệnh dưới dây ở trong thư mục của dự án
```
npm start
```
